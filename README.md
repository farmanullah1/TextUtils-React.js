**TextUtils Studio** is a premium, browser-based text manipulation and analytics toolkit. Built with a stunning "Deep Cyber Glass" UI, it offers over 100 advanced text formatting, extraction, and cryptography tools designed for developers, writers, and data analysts. 

The application is 100% client-side, meaning your data never leaves your browser, ensuring complete privacy and lightning-fast performance.

## ✨ Key Features (100+ Tools)

### 📊 Deep Live Analytics
* Real-time counters for Words, Characters, Sentences, and Paragraphs.
* Advanced insights: Data Size (Bytes), Total Lines, Characters (no spaces), Longest Word.
* Estimated Reading and Speaking time algorithms.

### 💻 Developer Formatting & Casing
* **Standard Casing:** UPPERCASE, lowercase, Title Case, Sentence case, aLtErNaTe, iNVERSE, rAnDoM.
* **Dev Casing:** `camelCase`, `PascalCase`, `snake_case`, `kebab-case`, `MACRO_CASE`, `dot.case`, `path/case`.
* **Code Tools:** Instant JSON stringify formatting and line trimming.

### 🧹 Deep Cleaning
* Instantly fix irregular spacing and remove empty lines.
* Strip HTML tags, brackets, accents, and punctuation.
* Isolate data by removing numbers, letters, vowels, consonants, or emojis.

### 🔍 Advanced AI Extraction Bot
* Extract specific data points instantly: Emails, URLs, Phone Numbers, IP Addresses, and MAC Addresses.
* Social Media: Extract `#hashtags` and `@mentions`.
* Development: Extract Hex Colors (`#FFFFFF`) and Dates.

### 🔐 Cryptography & Web Encoders
* Encode/Decode Base64, URL, and HTML entities.
* Convert text to Binary, Hexadecimal, and Octal.
* Ciphers: ROT13, Caesar Shift (+1 / -1), Atbash, Morse Code, and L33T Speak.

### 📝 Structure & Lists
* Remove duplicate lines/words.
* Sort alphabetically (A-Z / Z-A) or by string length.
* Line modifiers: Add bullets, numbers, dashes, quotes, or convert to CSV/TSV.
